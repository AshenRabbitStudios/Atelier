- 2026-07-11 **Context documents get a second write tool: `edit_<plugin>__<key>` (targeted find-and-replace) alongside `set_` (full replace).** Full-rewrite-only forces the agent to regenerate the _whole_ value to change any part of it â€” every untouched byte passes back through sampling, so paraphrase/detail drift is silent (the tool result shows no diff) and **compounds** (the mutated doc is next turn's injected ground truth; there is no stable substrate).
