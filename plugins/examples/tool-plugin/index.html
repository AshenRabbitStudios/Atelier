Tool Plugin — contributes agent tools

This is a kind: both plugin: a panel (this view) plus tools the agent can call. The tools run in an isolated backend process, not in this sandbox or the app.

reverse_text(text)
Reverse the characters of a string.
text: string
sum_numbers(a, b)
Add two numbers.
a: number, b: number

Try: ask the agent to "use the reverse_text tool on 'hologram'". Disable the plugin and the tools disappear on the next turn.
